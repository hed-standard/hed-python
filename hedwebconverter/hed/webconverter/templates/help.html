Using the Converter

The HED Converter is an online-tool designed to make creating new HED Schema easy. You can quickly convert between XML and mediawiki formats, while also ensuring your schema has no duplicate tags. All you need to start with is a source HED Schema in .xml or .mediawiki format. This can be a local file or from a URL.

Step 1: Specify the HED Schema you want to be converted.

You can either use a URL(make sure it is a direct download URL, e.g. the "raw" link on github) or you can upload a local file using the browse button. Accepted file types are .xml and .mediawiki.

Step 2a: Convert the HED Schema from XML/Mediawiki.

This action should quickly return a file to download containing the converted schema, assuming you used a valid schema.

Step 2b: Check for duplicate tags in the HED Schema

This action will check if you have any duplicate tags in your original file, and return a .txt file containing a list if any are present. Note: This implicitly converts .mediawiki versions to .xml versions for processing, so it will not work if your file cannot convert.
