For this error it seems to be a random place in the text of the tag it occurs in. That makes this error easier to quickly identify.

This is another common issue that occurs. The reason for the error is that tags are separated by commas, so when one is encountered in a descriptions text, or any text in a tag, it is considered the start of a new tag. Make sure to proofread your descriptions and text in tags, then remove any commas that are present from the text to fix this.
